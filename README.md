# 北京交通大学博士学位论文模板

作者： [@gzzang](https://github.com/gzzang)

## 说明

本模板仅适用于双面打印的北京交通大学博士学位论文，使用 XeTeX 引擎进行编译。
* 注意：非官方，且不适用于北京交通大学硕士学位论文

## 编译

可直接运行 compile.bat 文件。
